# B站运营策略

## 为什么选 B站

| 维度 | B站 | 抖音 | 
|------|-----|------|
| AI 内容审核 | ⚠️ 相对宽松 | 🔴 严 |
| 用户口味 | 接受猎奇/脑洞 | 大众娱乐 |
| AI 内容接受度 | 能接受 | 反感 |
| 粉丝粘性 | 高 | 低 |
| 创作激励 | ✅ 有分成 | ❌ |
| 导流容忍度 | ⚠️ 相对宽松 | 🚫 严打 |

## 发布策略

### 标题公式
`[题材标签] + [悬念/冲突] + [数字/对比]`

示例：
- 「男孕第1集：他醒来发现肚子大了」
- 「同事怀孕了，但他是男的」

### 标签体系
- 固定标签：`AI短剧` `AI漫剧` `AI视频`
- 题材标签：`悬疑` `反转` `脑洞`
- 系列标签：`男孕` `第X集`

### 发布节奏
- 试水期：每周 2 集，测哪集数据好
- 稳定期：固定 周二/周四 更新
- 最佳发布时间：工作日晚 19:00-21:00

### 评论区运营
- 首条评论置顶引导讨论（提问式）
- 回复前 30 条评论
- 下集预告在评论区而不是视频里（增加互动）

## 增长飞轮

```
发布视频 → 播放量数据 → 分析打开率/完播率/互动率
    → 优化下集剧本/标题/封面 → 数据提升 → 更多推荐
```

## 安全边界

| 能做 | 不能做 |
|------|--------|
| 脑洞/猎奇剧情 | 血腥/暴力/色情 |
| AI 标注「AI生成」 | 假装真人实拍 |
| 引导关注 | 引导外部平台（前3集不要） |
| 简介放爱发电 | 视频内容强推销 |
