# 提示词模板库

## 一、角色生成模板

```
[画质词] [角色描述] [脸部细节] [服装细节] [光照] [构图] [比例]

示例：
电影级真实风格。一位 28 岁亚洲男性，黑色短发，下颌线条分明，
深棕色眼睛。身穿白色衬衫，领口敞开。柔和的侧光从左侧打入。
半身中景，直视镜头，9:16 竖屏构图。
```

### 画质词库（选 1-2 个，全剧固定）
`电影级真实 | 高清写实 | 超写实渲染 | 真实照片风格`

### 光照词库（全剧固定）
`柔和的侧光 | 自然日光 | 暖黄室内光 | 清晨漫射光 | 三点布光`

## 二、场景生成模板

```
[场景类型] [主要元素] [光照] [氛围] [景别]

示例：
现代办公室。落地窗前一张原木办公桌，电脑屏幕亮着蓝色代码。
清晨阳光从百叶窗缝隙透入。安静压抑的氛围。中景，平视。
```

## 三、图生视频模板

```
[动作描述] [约束条件] [禁止项]

示例：
同一个角色缓慢转头面向镜头，嘴角微微上扬。
保持相同的面部特征、白色衬衫、柔和侧光、电影风格。
平滑控制动作。
禁止：服装变化、面部变形、大幅镜头运动。
```

### 安全动作词库（优先使用）
`缓慢转头 | 嘴唇微动 | 视线移动 | 头发轻飘 | 衣摆微摆 | 镜头缓推 | 轻微点头 | 眼睛眨动`

### 危险动作（高崩率，避免）
`转身冲刺 | 大幅度挥手 | 跑步 | 跳跃 | 多人互动 | 快速转头`

## 四、提示词审查清单

生成前逐条检查：
- [ ] 风格词是否全剧统一
- [ ] 角色外貌描述是否一字不变
- [ ] 无新风格词混入（动漫/水彩/赛博朋克等）
- [ ] 无模糊形容词（漂亮/好看/帅气）
- [ ] 动作是否为「安全动作」
- [ ] 光照/环境是否与同场景一致
