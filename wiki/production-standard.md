# AI 短视频创作规范

> **AI 漫剧不是一键生成的，是一段一段搭出来的。角色一致性不是写出来的，是管出来的。**

## 一、项目目录结构

```
project-name/
├── 01-世界观设定.md
├── 02-角色库/
│   ├── 主角-XXX/（角色卡/定妆照/三视图/表情集/服装库）
│   └── 场景库/
├── 03-剧本/（剧本/分镜表/提示词CSV/成片剪辑表）
├── 04-素材/（E01/静帧/视频/配音/字幕）
├── 05-成片/
└── 06-发布/
```

## 二、AI 友好型写作规则

**每条描述必须可直接转化为画面。** AI 不理解抽象叙事。

| ❌ | ✅ |
|----|-----|
| 他感到绝望 | 他低下头，眼眶泛红，右手攥紧衣角 |
| 办公室气氛紧张 | 日光灯闪烁，两人隔桌对视，无人说话 |

### 剧本格式标准
```
【场景】办公室·夜晚·室内
中景 | 暖黄顶光 | 45°俯角
人物：林晨 | 动作：盯着屏幕，眉头紧锁
表情：焦虑，嘴唇紧抿
时长：4秒
```

### 节奏公式
```
0-3秒   视觉钩子
3-15秒  展开冲突
15-25秒 反转/高潮
25-30秒 开放性结局
```
反转必须在第 10 秒左右出现。

## 三、提示词铁律

| 规则 | 说明 |
|------|------|
| 🔒 固定风格词 | 全剧统一基础词 |
| 🔒 固定角色描述 | 外貌描述全剧一字不变 |
| ❌ 禁加新风格词 | 不加「动漫」「赛博朋克」等 |
| ❌ 禁用模糊词 | 用「下颌分明」不用「帅气」 |
| ⚠️ 动作克制 | 「缓慢转头」>「转身冲刺」 |
| ⚠️ 环境一致 | 同场景固定光照描述 |

## 四、角色一致性三支柱

1. **视觉锚定 > 文字描述** — 每次生成都挂参考图
2. **动作克制** — 大动作崩脸，小动效稳定
3. **光照/环境一致** — 全剧统一光照方向

## 五、质量标准

逐镜验收：面孔一致 / 服装一致 / 光照一致 / 动作流畅 / 无AI穿帮
成片验收：人声清晰 / 字幕同步 / BGM不盖对白 / 前3秒有钩子 / 竖屏9:16

## 六、自动化流程

```
剧本 → 提示词 → API 生图/视频 → TTS → FFmpeg合成 → 成片
（全程服务器完成，你只审片+发布）
```
