# AI 友好型剧本写作规则

> **AI 看不懂「他很绝望」。你必须告诉 AI：他低下头，眼眶泛红，右手攥紧衣角。**

---

## 一、核心原则

### 1.1 剧本 ≠ 提示词

| 传统剧本 | AI 剧本 |
|----------|---------|
| 写心理活动 | 写可见动作 |
| 写氛围 | 写光照 + 景别 |
| 写「他经历了转折」 | 写「他停下脚步，回头看了一眼，眼神由迷茫变为坚定」 |
| 写对白为主 | 写视觉为主，对白为辅 |

### 1.2 铁律三条

| # | 规则 | 检查标准 |
|---|------|---------|
| 1 | **每句描述可画** | 闭上眼能想象出画面 = 合格 |
| 2 | **零抽象叙事** | 无「感到」「觉得」「似乎」「仿佛」 |
| 3 | **动作可执行** | 动作不超过安全词库范围 |

## 二、剧本格式标准

### 2.1 单镜头格式（强制字段）

```
【场景X】场景名 · 时间 · 室内/室外
景别 | 光照方向+类型 | 角度
人物：角色名
动作：具体物理动作（动词 + 身体部位）
表情：具体面部变化（眉毛/眼睛/嘴/面部肌肉）
对白（角色名）：「台词文本」
时长：X秒
```

**完整示例**：

```
【场景1】办公室 · 夜晚 · 室内
中景 | 暖黄顶光 + 蓝色屏幕光 | 45°俯角
人物：林晨
动作：右手食指悬在键盘上方，没有按下。喉结滚动一次。
表情：眉头紧锁，嘴唇紧抿成一条线，眼睛一眨不眨盯着屏幕。
对白（林晨）：「来不及了。」
时长：4秒
```

### 2.2 为什么字段必须全

| 字段 | AI 需要它是为了 |
|------|----------------|
| 景别 | 确定画幅（特写/中景/全景） |
| 光照 | 锁定光源方向和色温 |
| 角度 | 确定机位（俯角=压迫，仰角=权威） |
| 动作 | 告诉图生视频模型「让角色做什么」 |
| 表情 | 告诉图生视频模型「脸怎么动」 |
| 时长 | 决定视频片段切点 |

## 三、按类型剧本节奏公式

### 3.1 🎭 悬疑短剧（30 秒）

```
0-3秒    反常画面（不该出现的东西出现在不该在的地方）
3-8秒    主角发现异常（表情变化）
8-12秒   调查/逼近真相（快速切镜2-3个）
12-18秒  第一次反转（信息揭露）
18-24秒  第二次反转（推翻第一次）
24-28秒  危机升级（威胁逼近）
28-30秒  开放性留白（戛然而止）
```

**悬疑钩子模板**：
- 「一双眼睛在黑暗中睁开」
- 「手机屏幕亮起，显示一条三年前的消息」
- 「他数了数楼梯——多了一级」
- 「镜子里的人没有眨眼」

### 3.2 💕 甜宠短剧（30 秒）

```
0-3秒    意外交集（撞到/接到/对视）
3-8秒    微表情特写（嘴角/眼神/手指）
8-15秒   暧昧拉扯（靠近→后退→再靠近）
15-22秒  甜蜜点（一句台词/一个动作）
22-28秒  小阻碍/误会暗示
28-30秒  角色反应留白
```

**甜宠钩子模板**：
- 「他的手停在我脸颊前方一厘米」
- 「全公司都在看我们，但他没有松手」
- 「她说最讨厌我，但她的耳朵红了」

### 3.3 🚀 逆袭/爽文短剧（30 秒）

```
0-3秒    被打压的瞬间（羞辱/看不起）
3-8秒    冷静反应（不争辩，一个眼神/动作）
8-15秒   反转证据出现（真相/实力显现）
15-22秒  打脸高潮（对手表情变化）
22-28秒  主角离场/宣言
28-30秒  旁观者反应
```

**逆袭钩子模板**：
- 「全场都在笑他，直到他掏出那张黑卡」
- 「『这个项目没人能做完』——他把方案拍在桌上」
- 「她退婚那天，他成了首富」

### 3.4 😂 搞笑/吐槽短剧（20-30 秒）

```
0-3秒    日常场景 + 一句话建立预期
3-8秒    预期被打破（荒诞转折）
8-15秒   角色夸张反应
15-20秒  第二层荒诞（雪上加霜）
20-30秒  收尾吐槽（打破第四面墙可选）
```

**搞笑钩子模板**：
- 「谁说 AI 不会取代人类——老板刚用 ChatGPT 把我开了」
- 「面试官问我期望薪资，我报了——他笑了，我哭了」
- 「她以为我在跟她表白，其实我在念外卖菜单」

## 四、对白规则

### 4.1 TTS 配音约束

| 约束 | 原因 |
|------|------|
| 单句 ≤ 20 字 | TTS 单次生成极限 |
| 每镜最多 1 句对白 | 语速与视频时长匹配 |
| 不写语气词（嗯/啊/哦） | TTS 处理差 |
| 不写多人同时说话 | 后期合成困难 |
| 情绪标注用括号 | `（低声）「别走。」` |

### 4.2 对白密度

| 视频时长 | 对白总字数 | 对白镜头数 |
|---------|-----------|-----------|
| 15 秒 | ≤ 30 字 | 2-3 个 |
| 30 秒 | ≤ 60 字 | 4-6 个 |
| 60 秒 | ≤ 120 字 | 8-10 个 |

> ⚠️ **对白不是填满时长的工具。** 沉默 + 画面 > 啰嗦对白。

### 4.3 对白写作原则

| ✅ 好对白 | ❌ 差对白 |
|----------|----------|
| 「三年前的事，你还记得吗？」 | 「你还记得三年前那件事吗？那件事改变了一切……」 |
| （沉默，眼眶泛红） | 「我很难过，因为我……」（内心独白） |
| 「他没来。」 | 「他没有来，我很失望，因为他答应过我……」 |
| 「这个数字不对。」 | 「经过我仔细计算，这个数字存在三个错误……」 |

## 五、抽象 → 可视化翻译表

这是剧本写作中最关键的技能：**把所有抽象叙事翻译成视觉指令**。

| 抽象叙事 | → 可视化 | 所需镜头 |
|----------|---------|---------|
| 他感到绝望 | 他低下头，眼眶泛红，右手攥紧衣角 | 1 镜 |
| 办公室里气氛紧张 | 日光灯闪烁，两人隔桌对视，无人说话 | 2 镜：广角→两人近景 |
| 她经历了内心转折 | 她停下脚步，回头看了一眼，眼神由迷茫变为坚定 | 1 镜 |
| 时间流逝 | 窗外天色由暗变亮，桌上的咖啡杯空了又满 | 3 镜快切 |
| 他变强了 | 同角度、同光照——第一镜气喘吁吁，第二镜气定神闲 | 2 镜对比 |
| 城市很冷漠 | 雨中人流穿梭，无人多看摔倒的主角一眼 | 2 镜 |
| 她爱上他了 | 她把咖啡推到他面前——杯柄的方向转向了他 | 1 镜（细节） |
| 出大事了 | 手机从手中滑落，屏幕朝下摔在地上 | 1 镜 |
| 有人在监视 | 远处车窗反射出一个模糊的人影 | 1 镜 |
| 他很孤独 | 他坐在长桌一端，另一端是空的椅子 | 1 镜 |

## 六、每集剧本长度限界

| 目标成片 | 分镜数 | 每镜时长 | 总字数（剧本） |
|---------|--------|---------|--------------|
| 15 秒 | 4-6 | 2-5 秒 | ~200 字 |
| 30 秒 | 6-10 | 2-5 秒 | ~400 字 |
| 60 秒 | 10-16 | 3-6 秒 | ~800 字 |
| 90 秒 | 14-20 | 3-7 秒 | ~1200 字 |

> **经验：** 起步做 30 秒。15 秒来不及展开，60 秒生图量翻倍但完播率更低。

## 七、钩子设计模式

前 3 秒决定观众是划走还是留下。以下是 6 种验证有效的钩子类型：

| 类型 | 模式 | 示例 |
|------|------|------|
| 🔮 信息差 | 角色知道，观众不知道 | 「只有我知道，再过 3 小时世界就毁灭了」 |
| 😱 反常画面 | 不该出现的东西出现 | 办公桌上摆着一张自己的遗照 |
| 🎯 直接冲突 | 第一帧就是冲突 | 一巴掌打在脸上，没有前因 |
| ❓ 悬念提问 | 提出问题不回答 | 「你知道人死之前最后一秒在想什么吗？」 |
| 🔄 对比反差 | 两个极端同框 | 西装革履的男人在捡垃圾桶里的面包 |
| 🚪 未完成动作 | 动作在临界点暂停 | 手指悬在红色按钮上方 |

## 八、剧本 → 分镜表 → 提示词翻译

### 8.1 完整翻译链路

```
文学剧本（人读）→ 逐镜格式（AI 读）→ 逐镜提示词（API 读）
```

**文学剧本片段**：
> 林晨深夜在办公室发现系统被入侵，他试图修复但来不及了。他绝望地靠在椅子上。

**↓ 翻译为逐镜格式 ↓**

```
【镜1】办公室 · 夜晚
中景 | 蓝色屏幕光 | 平视
人物：林晨
动作：快速敲击键盘，手指突然停住
表情：瞳孔放大，嘴唇微张
时长：3秒

【镜2】同上场景
特写 | 蓝色屏幕光 | 45°俯角
人物：林晨
动作：缓慢向后靠在椅背上，右手从键盘滑落
表情：眉头紧锁，嘴唇紧抿，眼眶泛红
对白（林晨）：「来不及了。」
时长：4秒
```

**↓ 翻译为提示词 ↓**

```
[镜1 提示词]
正面：电影级写实。{林晨角色卡描述}。坐在电脑前快速敲击键盘，
手指突然停在半空。蓝色屏幕光从正面照亮面部。瞳孔放大。
中景，平视，9:16竖屏。
负面：blur, distort, low quality, warping fingers, jittery eyes, plastic skin

[镜2 提示词]
正面：电影级写实。{林晨角色卡描述}。缓慢向后靠在椅背上，
右手从键盘滑落。眉头紧锁，嘴唇紧抿，眼眶泛红。
蓝色屏幕光从正面照亮面部。特写，45°俯角，缓慢推近，9:16竖屏。
负面：blur, distort, low quality, warping fingers, jittery eyes, plastic skin, frozen lips
```

### 8.2 翻译检查清单

每完成一个镜头的「文学→提示词」翻译，检查：

- [ ] 抽象叙事是否全部消除（无「感到」「觉得」「似乎」）
- [ ] 角色描述是否从角色卡复制粘贴（一字不改）
- [ ] 动作是否为安全动作
- [ ] 光照是否与该场景其他镜头一致
- [ ] 风格词是否全剧统一
- [ ] 负面词是否 ≤ 8 个
- [ ] 景别 + 角度是否明确

## 九、常见写作反模式

| ❌ 反模式 | 问题 | ✅ 正确写法 |
|----------|------|-----------|
| 「他回想起过去」 | 闪回需要额外镜头 | 同场景完成，不闪回 |
| 「与此同时……」 | 交叉剪辑复杂 | 顺序叙事 |
| 「气氛凝重得像铅」 | AI 不理解比喻 | 「无人说话，时钟滴答声格外响」 |
| 「她美得不可方物」 | 不可视觉化 | 「路人都回头看她」 |
| 「战斗持续了十分钟」 | 无法表现时长 | 「他从地上爬起来，嘴角带血，再次握紧拳头」 |
| 「他做了个梦」 | 梦境场景视觉不统一 | 避免写梦境/幻觉 |
| 「内心 OS：……」 | 需要旁白轨 | 用表情和动作替代内心独白 |
| 一镜到底写三行动作 | 一个镜头只能做一件事 | 拆分镜头 |

## 十、第 N 集 → 第 N+1 集迭代循环

B 站发布后根据数据反馈改写剧本：

| 数据信号 | 剧本调整 |
|---------|---------|
| 前 3 秒流失率 > 60% | 换钩子类型 |
| 中段流失率高 | 加第 8-12 秒的反转 |
| 完播率高但互动低 | 加结尾悬念/提问 |
| 弹幕说「什么鬼」 | 检查叙事是否跳跃 |
| 弹幕说「脸崩了」 | 检查该镜头的参考图和动作复杂度 |
| 收藏率高 | 维持当前类型，不换赛道 |
