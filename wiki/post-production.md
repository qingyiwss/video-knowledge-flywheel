# 后期制作全流程

> **后期占成片质量的 40%。生成素材是原材料，后期是烹饪。**

---

## 一、字幕工作流

### 1.1 字幕生成链路

```
配音文件(.mp3) + 台词文本
         │
         ▼
    【自动生成 SRT】
    - Edge-TTS: --write-subtitles 直接出 VTT
    - 剪映: 自动语音识别
    - 手动: 按台词时间轴手写
         │
         ▼
    【SRT 格式检查】
    - 每条 ≤ 15 字（竖屏一行太多字）
    - 时间戳精确到 0.1 秒
    - 前后留 0.2 秒缓冲
```

### 1.2 Edge-TTS 生成字幕

```bash
# 生成配音 + VTT 字幕
edge-tts --text "你终于来了" --voice zh-CN-YunxiNeural \
  --write-media line.mp3 --write-subtitles line.vtt

# VTT 转 SRT（FFmpeg 一键）
ffmpeg -i line.vtt line.srt
```

### 1.3 字幕样式标准

```
竖屏字幕规范：
┌──────────────────┐
│                  │
│     👤画面区域    │ ← 人物面部始终在画面中上 1/3
│                  │
│ ┌──────────────┐ │
│ │ 字幕不超过15字 │ │ ← 白色字体 + 黑色描边
│ └──────────────┘ │
│                  │
└──────────────────┘

字体：思源黑体 / 阿里巴巴普惠体（免费可商用）
字号：画面宽度的 5-6%
位置：画面底部 10-15% 区域
描边：1px 黑色，确保任何背景都能看清
```

### 1.4 SRT 格式

```srt
1
00:00:00,000 --> 00:00:02,500
你终于来了。

2
00:00:02,800 --> 00:00:04,300
我等了你很久。

3
00:00:04,800 --> 00:00:06,200
（沉默）

4
00:00:06,500 --> 00:00:08,000
所以呢？
```

> ⚠️ 非对白声音也要字幕：（沉默）/（脚步声）/（电话铃响）

## 二、BGM & 音效

### 2.1 BGM 选曲原则

| 原则 | 说明 |
|------|------|
| 不盖对白 | BGM 音量在对白段的 -18dB 以下 |
| 情绪匹配 | 悬疑=低音弦乐，甜宠=钢琴，逆袭=鼓点 |
| 循环不违和 | 选 30 秒循环段，不选有高潮段的曲子 |
| 无版权风险 | 只用免费可商用音源 |

### 2.2 免费 BGM 来源

| 来源 | 类型 | 链接 |
|------|------|------|
| 剪映内置 | 全类型 | 剪映→音频→音乐 |
| YouTube 音频库 | 全类型 | youtube.com/audiolibrary |
| Pixabay Music | 全类型 | pixabay.com/music |
| Freesound | 音效 | freesound.org |

### 2.3 4 类题材 BGM 配方

| 题材 | BGM 类型 | 音色特点 | BPM |
|------|---------|---------|-----|
| 悬疑 | Dark Ambient / Cinematic Tension | 低音、弦乐、电子嗡鸣 | 60-80 |
| 甜宠 | Lo-fi / Acoustic Piano | 钢琴、吉他、轻鼓点 | 80-100 |
| 逆袭 | Epic / Orchestral Build | 弦乐渐强、定音鼓 | 100-130 |
| 搞笑 | Quirky / Jazz / Funk | 拨弦、管乐、轻快打击 | 100-120 |

### 2.4 BGM 编辑要点

```
开头 0-2 秒：   BGM 渐入（避免突然炸耳）
对白段：        BGM 降到 -18dB
情绪转折点：    BGM 换段或变奏（标记情绪变化）
结尾 2 秒：     BGM 渐出
无对白段：      BGM 提到 -10dB
```

### 2.5 音效使用

| 场景 | 音效 | 作用 |
|------|------|------|
| 紧张 | 心跳声（低频） | 制造压迫感 |
| 发现 | 轻微的「叮」 | 提示关键信息 |
| 转折 | 弦乐短促一刺 | 强调反转 |
| 结束 | 回声/混响尾音 | 余韵 |

> **铁律：一集 30 秒，音效不超过 3 个。多了像搞笑视频。**

## 三、剪辑工作流

### 3.1 素材准备清单

```
04-素材/E01/
├── 静帧/（各镜头 AI 生成的定帧图）
├── 视频/（各镜头 AI 生成的视频片段）
├── 配音/（TTS 生成的配音文件）
├── 字幕/（SRT 字幕文件）
├── BGM/（背景音乐）
└── 音效/（音效文件）
```

### 3.2 剪映时间轴模板

```
音轨2: BGM ──────────────────────────────────────────────
音轨1: 音效      ██         ██
音轨0: 对白  ██    ██  ██  ████  ██  ██
视频轨: 镜1 │镜2│镜3│镜4│镜5│镜6│镜7│镜8│镜9│镜10
字幕轨: 字1│字2│字3│字4│字5│字6│字7│字8│字9│字10
       ═══════════════════════════════════════
       0s                   15s                 30s
```

### 3.3 剪辑顺序

```
1. 导入所有视频片段 → 按分镜顺序排列
2. 粗剪：每段卡到分镜表标注的时长
3. 导入配音文件 → 对齐到对应镜头的视频轨道
4. 导入字幕 SRT → 对齐到配音
5. 加 BGM → 调节音量曲线
6. 加转场（可选）
7. 加音效（2-3 个关键位置）
8. 导出前预览一遍 → 微调
9. 导出：1080p、30fps、9:16 竖屏
```

### 3.4 转场规则

| 转场 | 时长 | 使用位置 |
|------|------|---------|
| 硬切 | 0 | 主力（80% 镜头） |
| 叠化 | 0.5s | 时间流逝/场景切换 |
| 闪白 | 0.2s | 回忆/震惊/反转 |
| 黑屏 | 1s | 章节分隔/结尾留白 |

> **AI 短视频最多用 1-2 个特效转场。过多 = 业余。**

## 四、缩略图设计

### 4.1 为什么缩略图决定生死

B 站推荐流中，缩略图是唯一决定点击的视觉元素。

| 要素 | 占比 |
|------|------|
| 缩略图 | 70% 的点击决策 |
| 标题 | 25% |
| 播放量/弹幕 | 5% |

### 4.2 缩略图 4 要素

```
┌────────────────────┐
│  😱 表情放大        │ ← 面部占比 40%+ 区域
│  （惊讶/恐惧/笑容）  │
│                    │
│  【3-5 字大字标题】  │ ← 黄色/红色大字，白底黑边
│                    │
│   小字铺垫（8字内）  │ ← 左下角或右下角
└────────────────────┘
```

### 4.3 缩略图公式

| 题材 | 构图公式 | 表情 | 配色 |
|------|---------|------|------|
| 悬疑 | 半张脸 + 暗背景 + 红色文字 | 恐惧/惊讶 | 黑+红+白 |
| 甜宠 | 双人头部 + 粉色背景 + 白色文字 | 微笑 | 粉+白+金 |
| 逆袭 | 仰拍角色 + 暗金背景 + 黄色大字 | 冷峻/傲视 | 黑+金+白 |
| 搞笑 | 夸张表情 + 黄色大字 + 红色箭头 | 震惊/苦笑 | 黄+红+白 |

### 4.4 AI 生成缩略图提示词

```
【悬疑】
电影级写实。{主角描述}。面部特写，面部占比画幅 60%，
右眼在阴影中，左眼瞪大直视镜头，嘴唇微张。
暗黑背景，左侧一道红色霓虹光。9:16竖屏。
负面: blur, distort, low quality, warm lighting, overexposed

【甜宠】
柔焦电影风格。{男女主描述}。两人面部特写，额头相抵，
间距 3cm，女主嘴角上扬，男主侧脸看女主。
粉色柔焦背景，金色光斑散落。9:16竖屏。
负面: blur, distort, low quality, harsh shadows, cold lighting
```

## 五、成片导出规格

| 参数 | 值 |
|------|-----|
| 分辨率 | 1920×1080（横）或 1080×1920（竖） |
| 帧率 | 30fps |
| 编码 | H.264 |
| 码率 | 8-12 Mbps |
| 音频 | AAC, 192kbps, 48kHz |
| 格式 | MP4 |

### FFmpeg 合成（不经过剪映的全自动方案）

```bash
# 将所有视频片段 + 配音 + 字幕合成为一个 MP4
ffmpeg \
  -i shot01.mp4 -i shot02.mp4 ... \
  -i dub.mp3 -i bgm.mp3 \
  -vf "concat,subtitles=subtitle.srt" \
  -c:v libx264 -crf 20 -preset medium \
  -c:a aac -b:a 192k \
  -r 30 -s 1080x1920 \
  E01_FINAL.mp4
```

## 六、成片终极验收清单

### 6.1 视觉验收

- [ ] 逐镜头角色面孔一致（对比母图）
- [ ] 无 AI 穿帮（手指/背景/文字变形）
- [ ] 景别有变化（无连续 3 镜同景别）
- [ ] 9:16 竖屏构图正确
- [ ] 前 3 秒有视觉钩子
- [ ] 结尾有留白/悬念

### 6.2 音频验收

- [ ] 所有人声清晰可辨
- [ ] BGM 不盖对白（对白段 -18dB 以下）
- [ ] 无突然音量跳变
- [ ] 无底噪/电流声
- [ ] 音效位置准确

### 6.3 字幕验收

- [ ] 字幕与语音完全同步（误差 < 0.2 秒）
- [ ] 每行 ≤ 15 字
- [ ] 字体清晰可读
- [ ] 非对白也有字幕标注

### 6.4 发布前检查

- [ ] 缩略图上传
- [ ] 标题 ≤ 20 字，含关键词
- [ ] 标签 5-8 个
- [ ] 简介 1-2 句，含引导互动
- [ ] 分区选择正确
