# 角色设计方法论 v2.0

> **AI 不认「帅」，只认「下颌线条分明、鼻梁高挺、眉骨突出」。角色设计的第一原则：用 AI 能稳定复现的特征描述。**

---

## 一、AI 友好型设计铁律

| # | 规则 | 原因 |
|---|------|------|
| 1 | **每个特征必须可量化** | AI 不理解审美，只理解几何 |
| 2 | **选 3 个「锚点特征」** | AI 靠锚点锁脸，不是靠整体印象 |
| 3 | **对称优先于个性** | 不对称脸型/发型崩率 ×3 |
| 4 | **避免「过渡色」** | 深棕→浅棕→金发这类渐变描述容易漂移 |
| 5 | **服装=第二张脸** | 远镜头靠服装识别角色，不靠脸 |
| 6 | **一个角色 ≤ 3 套服装** | 服装越多，一致性越难保持 |

## 二、角色视觉原型库

### 2.1 脸型坐标

```
圆脸 ←──────────────────────────→ 方脸
  1         2         3         4         5
(鹅蛋)    (心形)    (椭圆)    (长方)    (国字)

适合 AI：
  ✅ 类型 1-2（鹅蛋/心形）：AI 稳定率最高
  ✅ 类型 3（椭圆）：通用，适合主角
  ⚠️ 类型 4（长方）：需要精确颧骨描述
  ❌ 类型 5（国字/极方）：下颌角 AI 容易偏移
```

**描述模板**：
```
{形状}脸型，{颧骨描述}，{下颌描述}，{下巴描述}
示例：「鹅蛋脸型，颧骨微隆但不突出，下颌线条柔和，下巴微尖」
```

### 2.2 眼型库

| 类型 | 描述 | 适合角色 | AI 稳定度 |
|------|------|---------|----------|
| 丹凤眼 | 眼尾上挑，内眼角下勾 | 冷峻、反派、御姐 | ⭐⭐⭐ |
| 桃花眼 | 眼尾微弯，卧蚕明显 | 温柔、甜宠男主 | ⭐⭐ |
| 杏眼 | 圆而大，双眼皮明显 | 女主、清纯型 | ⭐⭐⭐ |
| 细长眼 | 眼裂窄，单眼皮 | 冷静、智慧型 | ⭐⭐⭐⭐ |
| 下垂眼 | 眼尾下垂，无辜感 | 温柔、悲情 | ⭐⭐ |

> ⚠️ 避免描述：「好看的眼睛」「有神的眼睛」→ 无用。

**描述模板**：
```
{眼型}，{单/双}眼皮，{眼裂大小}，{眼尾走向}，{瞳色}
示例：「细长眼，单眼皮，眼裂中等，眼尾微微上挑，深棕色瞳孔」
```

### 2.3 鼻型库

| 类型 | 描述 | AI 稳定度 |
|------|------|----------|
| 直鼻 | 鼻梁直挺，鼻头适中 | ⭐⭐⭐⭐ |
| 翘鼻 | 鼻尖微翘，鼻梁中等 | ⭐⭐⭐ |
| 鹰钩鼻 | 鼻梁高，鼻尖下弯 | ⭐⭐ |
| 圆鼻 | 鼻头圆润，鼻翼略宽 | ⭐⭐⭐ |

**描述模板**：
```
{鼻梁高度}，{鼻梁形状}，{鼻尖方向}，{鼻翼宽度}
示例：「鼻梁高挺，鼻梁线条笔直，鼻尖微尖，鼻翼适中」
```

### 2.4 嘴型库

| 类型 | 描述 | 适合 |
|------|------|------|
| 薄唇 | 上下唇均薄，唇线清晰 | 冷静、刻薄型 |
| 厚唇 | 上下唇丰满 | 性感、热情型 |
| M唇 | 上唇呈M形曲线 | 甜宠女主 |
| 嘴角下垂 | 自然状态下嘴角向下 | 忧郁、严肃型 |

**描述模板**：
```
{唇形}，{唇厚}，{唇色}，{嘴角走向}
示例：「M形上唇，中等厚度，自然淡粉色，嘴角自然微扬」
```

## 三、角色 3 锚点法

### 原理

AI 每次重建角色，不会记住整体，但会记住 **3 个最突出的特征**。设计角色时精选 3 个锚点：

| 锚点位置 | 示例 | 权重 |
|---------|------|------|
| 发型 + 发色 | 「黑短发，三七分，右侧剃短」 | ⭐⭐⭐⭐⭐ |
| 标志物 | 「左手腕银色机械表」「左眉尾一道细疤」 | ⭐⭐⭐⭐ |
| 服装剪影 | 「白色立领衬衫永远敞开第一颗扣」 | ⭐⭐⭐⭐ |
| 面部特征组合 | 「细长眼+高直鼻+薄唇」= 冷峻型 | ⭐⭐⭐ |
| 体型特征 | 「宽肩窄腰，身高 185」 | ⭐⭐⭐ |

> **铁律：锚点按「远→中→近」分层。远景看发型+体型+服装剪影，中景看标志物+面部轮廓，特写看五官细节。**

### 实战：角色卡锚点区

```
【锚点1：发型】黑色短发，三七分，右侧耳上剃短至 3mm
【锚点2：标志物】左眉尾上方一道 1cm 细疤，斜向左上方
【锚点3：体态】宽肩窄腰，站立时微驼，右肩略高于左肩
```

## 四、体型 & 年龄坐标

### 4.1 体型坐标

```
体型 = {身高} + {肩宽} + {体脂} + {比例}

女性常见体型：
- 沙漏型：肩宽≈臀宽，腰极细 → 甜宠女主
- 矩形：肩宽≈臀宽≈腰 → 运动型
- 梨形：肩窄臀宽 → 亲和型
- 倒三角：肩宽＞臀宽 → 强势型

男性常见体型：
- 倒三角：宽肩，胸肌可见，腰窄 → 男主/武将
- 矩形：肩腰臀基本等宽 → 普通人/程序员
- 椭圆：腰围＞肩宽 → 喜剧角色/长辈
- 瘦高型：肩窄，四肢修长 → 学者/病弱型
```

**描述模板**：
```
身高{cm}，{体型}，{肩描述}，{腰描述}，{特征}
示例：「身高 178cm，倒三角体型，肩宽约 48cm，腰窄，右手比左手稍粗」
```

### 4.2 年龄标记

| 年龄段 | 面部特征 | 体态特征 |
|--------|---------|---------|
| 18-25 | 皮肤光滑，下颌线条清晰，无皱纹 | 体态紧致，动作轻快 |
| 26-35 | 轻微法令纹，眼下可能出现细纹 | 体态成熟，动作稳重 |
| 36-45 | 额头细纹，眼角鱼尾纹初现，肤色微暗 | 肩略前倾，动作有分量 |
| 46-60 | 法令纹深，眼袋可见，两鬓斑白可选 | 体态松弛，关节动作慢 |
| 60+ | 皱纹明显，皮肤松弛，头发花白 | 驼背，步伐慢 |

**AI 要注意**：
- ❌ 不要写「28 岁」就完——AI 对年龄理解不稳定
- ✅ 必须配合具体体征：「眼下无细纹，皮肤紧致有光泽，无抬头纹」

## 五、服装设计原则

### 5.1 服装 = 第二张脸

远景/背面镜头下，**观众靠服装识别角色**，不是靠脸。

| 原则 | 说明 |
|------|------|
| 每套服装必须有一个**标志元素** | 特殊领型/纽扣/配饰/图案 |
| 颜色固定 + 精确 | 「深藏青色」不是「蓝色」 |
| 同角色不同服装要有**统一元素** | 都带银色纽扣 / 都同一色系 |
| 避免纯黑/纯白 | AI 纯色容易脏或过曝 |

### 5.2 服装描述模板

```
{上装颜色+款式+材质+细节}，{下装颜色+款式}，{标志元素}
示例：「白色尖领长袖棉质衬衫，银色圆形纽扣共 5 颗领口第一颗敞开，
      黑色直筒西裤，左手腕银色钢带机械表」
```

## 六、角色风格坐标

```
写实 ←──────────────────────────→ 风格化
  1          2          3          4
(照片级)   (电影写实)  (半写实)   (动漫风)

AI 短视频推荐：2（电影写实）
- 位置 1（照片级）：容易 Uncanny Valley
- 位置 2（电影写实）：细节够又不假 ← 主力
- 位置 3（半写实）：介于之间的安全区
- 位置 4（动漫风）：需要模型支持，崩率波动大
```

**全剧统一选择，不可混用。**

## 七、角色关系设计

### 视觉差异化公式

多角色时，确保**任何两个角色在以下至少 3 个维度上不同**：

| 维度 | 角色 A | 角色 B |
|------|--------|--------|
| 身高差 | 185cm | 162cm |
| 体型差 | 倒三角 | 沙漏型 |
| 发型差 | 黑色短发 | 棕色长发马尾 |
| 服装色系 | 白+黑 | 米色+卡其 |
| 姿势语言 | 挺直 | 微驼 |
| 标志物 | 机械表 | 银色细链 |

> ⚠️ **AI 最怕「两个身高体型相仿的男性穿同色系衣服」**——这是 face swap 的高发场景。

## 八、完整角色卡模板

```markdown
## 角色卡：林晨

### 基本信息
- 姓名：林晨
- 性别：男
- 年龄：28 岁
- 身高：178cm
- 体重：72kg

### 锚点（3 个）
1. 黑色短发，三七分，右侧耳上剃短 3mm
2. 左眉尾上方一道 1cm 细疤
3. 白色尖领衬衫，银色圆扣，领口永远敞开第一颗

### 面部
- 脸型：鹅蛋脸，颧骨微隆，下颌线条分明，下巴微尖
- 眼：细长眼，单眼皮，眼裂中等，眼尾微挑，深棕色
- 鼻：鼻梁高挺，线条笔直，鼻尖微尖，鼻翼适中
- 嘴：薄唇，上唇M形曲线，自然淡粉，嘴角自然微扬

### 体态
- 体型：倒三角，肩宽约 48cm，腰窄
- 姿态：站立挺直，走路右肩微微前倾

### 服装（3 套）
1. **日常**：白色尖领长袖棉质衬衫 + 黑色直筒西裤 + 银色机械表
2. **外出**：深藏青色立领夹克 + 白色T恤 + 黑色牛仔裤
3. **正装**：深灰色三件套西装 + 银色袖扣

### 表情特征
- 平静时：嘴角微扬，像随时在微笑
- 紧张时：右眉会不自觉地微微上挑
- 愤怒时：下颌咬紧，颧骨更突出

### 风格坐标
电影写实风格（位置 2）
```

## 九、角色设计验证清单

设计完角色卡后逐条检查：

- [ ] 有 3 个明确锚点（发型/标志物/服装剪影）
- [ ] 所有描述可量化（无「好看」「帅气」「有气质」）
- [ ] 脸型 + 五官 = 6 项全覆盖
- [ ] 发型明确到分线/长度/特殊处理
- [ ] 每套服装有标志元素
- [ ] 身高体型有具体数字
- [ ] 与同剧其他角色 ≥ 3 个维度不同
- [ ] 年龄有具体体征，不是只有一个数字
- [ ] 风格坐标已锁定
