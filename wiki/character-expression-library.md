# 角色表情 & 姿态库

> **AI 短视频的表达靠「脸+身体」，不靠台词。每个情绪必须翻译为精确的面部肌肉动作和肢体语言。**

---

## 一、7 基础表情（AI 可执行版）

### 1.1 喜悦 😊

**面部**：
- 眉毛：自然或微上扬
- 眼睛：眼裂微眯，眼角出现细纹
- 嘴：嘴角上扬，牙齿可能露出
- 脸颊：苹果肌上提

**AI 提示词**：
```
嘴角上扬，眼睛微眯，眼角出现笑纹，脸颊上提
```

**强度分级**：
| 级别 | 描述 | 适用 |
|------|------|------|
| 微笑 | 嘴角微扬，眼睛微弯 | 礼貌、含蓄 |
| 露齿笑 | 嘴角大幅上扬，上排牙齿露出 | 开心、友善 |
| 大笑 | 嘴张开，眼眯成缝，头可能后仰 | 极度喜悦 |

---

### 1.2 悲伤 😢

**面部**：
- 眉毛：内侧上扬，外侧下压（八字眉）
- 眼睛：眼裂缩小，眼睑下垂，眼眶泛红
- 嘴：嘴角下垂，下唇可能颤抖
- 脸颊：面部肌肉整体下坠

**AI 提示词**：
```
眉毛内侧上扬外侧下垂，眼眶泛红，眼睑沉重下垂，
嘴角向下，面部肌肉松弛下坠
```

**强度分级**：
| 级别 | 描述 |
|------|------|
| 伤感 | 眼眶微红，嘴唇微抿，眉毛微蹙 |
| 落泪 | 眼眶泪水打转，嘴唇颤抖 |
| 崩溃 | 泪流满面，面部扭曲，可能在抽泣 |

---

### 1.3 愤怒 😠

**面部**：
- 眉毛：眉头紧锁，压低
- 眼睛：瞪大或眯起，眼神锐利
- 嘴：嘴唇紧抿或咬牙切齿
- 其他：下颌咬紧，颧骨突出，鼻孔可能张大

**AI 提示词**：
```
眉头紧锁压低，眼睛微微眯起眼神锐利，嘴唇紧抿成一条线，
下颌咬紧，颧骨突出
```

> ⚠️ 愤怒是 AI 最容易崩的表情。避免「大吼」「咆哮」——嘴张大 = 崩脸概率 ×5。用「嘴唇紧抿+眼神+咬肌」表达。

---

### 1.4 恐惧 😨

**面部**：
- 眉毛：上扬 + 拉近
- 眼睛：瞳孔放大，眼白露出增多
- 嘴：嘴唇微张或紧抿，嘴角下拉
- 其他：面部肌肉紧绷，可能出汗

**AI 提示词**：
```
眉毛上扬拉近，瞳孔放大，嘴唇微张，面部肌肉紧绷，
额头可能出现细汗
```

---

### 1.5 惊讶 😲

**面部**：
- 眉毛：大幅上扬
- 眼睛：瞪大，瞳孔缩小
- 嘴：张开，O 形
- 其他：面部肌肉整体上提

**AI 提示词**：
```
眉毛大幅上扬，眼睛瞪大瞳孔缩小，嘴巴张开呈O形，
面部肌肉上提
```

> ⚠️ 惊讶和恐惧的区别：恐惧眉毛内聚，惊讶眉毛外扩。这俩容易混，AI 尤甚。

---

### 1.6 厌恶 😖

**面部**：
- 眉毛：眉头紧锁压低
- 眼睛：眼裂缩小，眼神回避
- 鼻：鼻梁皱起
- 嘴：上唇上提，嘴角下拉

**AI 提示词**：
```
眉头紧锁压低，鼻梁皱起，上唇上提露出上排牙齿边缘，
眼角微眯，眼神回避
```

---

### 1.7 轻蔑/不屑 😏

**面部**：
- 眉毛：一侧微挑
- 眼睛：眼裂微眯，视线斜向下
- 嘴：单侧嘴角上扬（不对称）
- 头：微微后仰或偏转

**AI 提示词**：
```
一侧眉毛微挑，眼睛微眯视线斜向下，单侧嘴角上扬，
头部微微后仰
```

> 这是最省镜头的表情——1 秒就能传达角色态度。

## 二、微表情序列

AI 短视频（2-5 秒/镜）中，微表情比大表情更常用、更稳定。

### 2.1 常用微表情

| 微表情 | 面部动作 | 含义 | 时长 |
|--------|---------|------|------|
| 眨眼放慢 | 闭眼 0.5 秒 → 睁开 | 思考、犹豫、说谎 | 2 秒 |
| 瞳孔微缩 | 瞳孔缩小，无其他变化 | 警觉、发现异常 | 1 秒 |
| 喉结滚动 | 吞咽动作 | 紧张、压抑 | 1 秒 |
| 嘴角抽动 | 单侧嘴角微微上扬 0.3 秒 | 讽刺、忍住笑 | 1.5 秒 |
| 眉头一蹙 | 眉头短暂锁紧后松开 | 疑惑、不认同 | 1.5 秒 |
| 视线飘移 | 视线从 A 点移向 B 点 | 逃避、思考 | 2 秒 |
| 嘴唇微抿 | 上下唇轻合加压 | 克制、忍耐 | 2 秒 |

### 2.2 情绪转折（一个镜头内的变化）

```
过渡公式：{起始表情} → {中间微表情} → {结束表情}

示例：轻蔑 → 震惊
「一侧嘴角上扬，眼睛微眯（起始）→ 
  嘴角缓慢回落，眉毛开始上扬（过渡0.5秒）→ 
  嘴巴张开，眼睛瞪大（结束）」
  
总时长：3-4 秒
```

## 三、肢体语言库

### 3.1 站姿语言

| 姿态 | 描述 | 传达 |
|------|------|------|
| 挺直 | 脊柱直立，双肩后展，下巴微抬 | 自信、强势、准备战斗 |
| 微驼 | 肩部前倾 10-15°，头微低 | 疲惫、防御、思考中 |
| 斜靠 | 重心在一只脚，身体靠墙/桌 | 放松、不设防、闲聊 |
| 抱臂 | 双臂在胸前交叉 | 防御、拒绝、保持距离 |
| 插手 | 双手插兜，肩放松 | 从容、漫不经心 |
| 握拳垂手 | 双臂自然下垂，双拳紧握 | 压抑愤怒、即将爆发 |

### 3.2 手势语言

| 手势 | 描述 | 传达 |
|------|------|------|
| 手指轻敲 | 食指/中指在桌面快速点击 | 不耐烦、思考、紧张 |
| 手掌摊开 | 手掌朝上，五指自然伸展 | 坦诚、无奈、提问 |
| 食指指向 | 食指伸出指向对方/物体 | 指责、强调、命令 |
| 五指张开压桌 | 五指张开按在桌面上 | 施压、掌控、压抑愤怒 |
| 攥紧衣角 | 一只手攥住自己的衣角/袖口 | 紧张、不安、隐忍 |
| 摩擦指节 | 拇指摩擦食指指节 | 焦虑、犹豫、思考 |

### 3.3 头部语言

| 动作 | 描述 | 传达 |
|------|------|------|
| 微微点头 | 头部小幅度上下 | 认同、鼓励对方说 |
| 缓慢摇头 | 头部左右缓慢摇动 | 否定、失望、不信 |
| 头偏转 | 头转向一侧 15-30° | 疑问、审视、不信任 |
| 抬头 | 下巴上扬，视线向上 | 回忆、思考、傲慢 |
| 低头 | 下巴贴近胸口 | 羞愧、认输、回避 |

## 四、AI 短视频「安全姿态」Top 10

> ⚠️ 复杂姿态（手交叉背后、盘腿坐、大幅度转身）= 变形概率高。优先用这 10 个安全姿态。

| # | 姿态 | 变形风险 | 适合场景 |
|---|------|---------|---------|
| 1 | 站立，双臂自然下垂 | 极低 | 日常对话 |
| 2 | 坐在桌前，双手放桌面 | 极低 | 办公场景 |
| 3 | 靠墙站立，单腿微曲 | 低 | 等待、观察 |
| 4 | 坐在椅子上，身体前倾 | 低 | 紧张对话 |
| 5 | 一只手举杯（咖啡/水） | 低 | 日常生活 |
| 6 | 双手插兜站立 | 低 | 从容、聊天 |
| 7 | 单手扶栏杆/窗框 | 低 | 看窗外 |
| 8 | 坐在沙发上，身体后靠 | 中 | 放松场景 |
| 9 | 双臂胸前交叉 | 中 | 防御姿态 |
| 10 | 一只手摸下巴（思考） | 中 | 思考、审视 |

## 五、情绪弧线设计

### 5.1 单集 30 秒情绪曲线

```
紧张度
  │     ╱╲
  │    ╱  ╲    ╱╲
  │   ╱    ╲  ╱  ╲
  │  ╱      ╲╱    ╲
  │ ╱              ╲___
  └──────────────────────→ 时间
  0s    8s    15s   22s  30s
```

每个波峰对应一次情绪变化，每次变化需要 1-2 个面部镜头。

### 5.2 4 类题材情绪配方

| 题材 | 情绪路径 | 镜头数 |
|------|---------|--------|
| 悬疑 | 平静 → 疑惑 → 恐惧 → 决心 | 4-5 镜 |
| 甜宠 | 疏离 → 好奇 → 心动 → 甜蜜 | 4-5 镜 |
| 逆袭 | 压抑 → 冷静 → 反击 → 傲视 | 4-5 镜 |
| 搞笑 | 日常 → 荒诞 → 夸张反应 → 吐槽 | 3-4 镜 |

### 5.3 情绪切换节奏

```
慢→快→慢 节奏：
- 第 1 个情绪状态 2-3 秒（建立基线）
- 转折微表情 0.5-1 秒（关键帧）
- 第 2 个情绪状态 2-3 秒（稳定新态）

快→快 节奏（高能场景）：
- 每个情绪状态 1-2 秒
- 转折 0.3 秒
- 适合：动作、惊恐、喜剧爆点
```
