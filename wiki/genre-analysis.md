# AI 短视频题材分析

> **选错题材 = 所有生产投入白费。在动手前花时间选对赛道，是最便宜的投资。**

### 1.1 6 大赛道

| 赛道 | 代表内容 | 难度 | AI友好度 | 变现潜力 |
|------|---------|------|---------|---------|
| 🎭 悬疑惊悚 | 都市怪谈、规则怪谈、细思极恐 | ⭐⭐ | ⭐⭐⭐⭐⭐ | ⭐⭐⭐⭐ |
| 💕 甜宠言情 | 霸总、先婚后爱、伪兄妹 | ⭐⭐ | ⭐⭐⭐ | ⭐⭐⭐⭐⭐ |

| 📖 真人叙事 | 真实故事改编、情感独白 | ⭐ | ⭐⭐ | ⭐⭐⭐ |

### 1.2 AI 友好度 = 什么题材 AI 能做好

| 维度 | ✅ AI 擅长的 | ❌ AI 不擅长的 |
|------|-----------|-------------|
| 角色数 | 1-3 个 | 5+ 人群像 |
| 场景 | 固定 2-3 个场景 | 每集换新场景 |
| 动作 | 站/坐/走/表情 | 打斗/舞蹈/运动 |
| 情绪 | 紧张、悲伤、冷漠 | 狂喜、大哭、极度愤怒 |
| 时长 | 15-30 秒/集 | 60 秒+ 长叙事 |

## 二、4 个推荐入门题材深度拆解

### 2.1 🎭 悬疑惊悚（⭐推荐入门）

**为什么选它**：
- AI 最擅长的情绪是「冷」「静」「暗」
- 悬疑靠表情和氛围，不靠大动作
- 观众对「AI 感」容忍度最高——甚至加分

**适合选题**：
```
✅ 规则怪谈：「深夜收到一条短信：天黑后不要照镜子」
✅ 细思极恐：「公司新来的同事和我死去的朋友一模一样」
✅ 都市传说：「这栋楼的 4 层从来不停电梯」
✅ 心理惊悚：「我能看到每个人剩下的寿命——包括我自己」
```

**单集结构**：反常→调查→反转→留白（30 秒）
**角色数**：1-2 个
**场景数**：1-2 个

### 2.2 💕 甜宠言情（⭐推荐入门+变现）

**为什么选它**：
- B 站女性用户付费意愿最强
- 角色交互简单（2 人为主）
- 变现路径清晰（爱发电番外/加更）

**适合选题**：
```
✅ 霸总：「全公司都怕他，只有我不知道他是老板」
✅ 伪兄妹：「我妈再婚后，我多了个全校最帅的哥哥」
✅ 先婚后爱：「协议结婚第一天，他把我堵在墙角」
✅ 穿越：「醒来发现我是恶毒女配——但男主好像爱上我了」
```

**单集结构**：意外交集→微表情→拉扯→甜蜜点（30 秒）
**角色数**：2 个
**场景数**：2-3 个

### 2.3 🚀 逆袭打脸（⭐推荐入门）

**为什么选它**：
- 节奏最好控制（压抑→反转→打脸，三步走）
- 情绪曲线上扬，完播率高
- AI 做冷脸/傲视表情极稳

**适合选题**：
```
✅ 赘婿逆袭：「三年后他归来，会议室所有人站了起来」
✅ 重生：「她退婚那天，他成了首富」
✅ 扮猪吃虎：「全班都笑他穷——他掏出一张黑卡」
✅ 王者归来：「被开除的第 100 天，前公司求他回去」
```

**单集结构**：被打压→冷反应→反转物→打脸（30 秒）
**角色数**：1-2 个
**场景数**：2 个

### 2.4 避坑题材

| 题材 | 问题 |
|------|------|
| 🎬 古装 | 服装/场景复杂度高，AI 崩率高 |
| ⚔️ 战斗/动作 | AI 做不出连续动作，像 PPT |
| 👥 群像校园 | 4+ 角色 = face swap 灾难 |
| 🔬 硬科幻 | 特殊视觉资产（飞船/机甲）AI 不稳定 |
| 😭 虐恋大哭 | 哭戏是 AI 崩率最高的表情 |

## 三、题材选择决策矩阵

### 3.1 3 个核心问题

| 问题 | 为什么重要 |
|------|-----------|
| 这个题材的观众在 B 站吗？ | 在错误的平台做对的内容 = 白做 |
| 这个题材 AI 能做到 70 分吗？ | 做不到 70 分就别做 |
| 这个题材我能持续产出吗？ | 短剧是连续剧，至少 10 集起 |

### 3.2 决策公式

```
选题材 = 用户需求 ∩ AI 能力 ∩ 个人可持续产出

悬疑惊悚 + 甜宠言情 → AI 友好度最高 + 变现最强
逆袭打脸 → AI 最容易 + 完播率最高
```

### 3.3 验证方法

做出 1 集样片发 B 站，看 3 个数据：

| 数据 | 达标线 | 不达标 = 换题材 |
|------|--------|----------------|
| 前 3 秒留存 | > 50% | 钩子/缩略图不行 |
| 完播率 | > 30% | 内容不对 |
| 点赞率 | > 3% | 共鸣不够 |

## 四、竞品拆解框架

分析 B 站同类 AI 短剧 UP 主：

```
UP主: [名称]
题材: [悬疑/甜宠/逆袭]
粉丝: X万
平均播放: X万
典型单集结构: [拆解前 10 秒]
弹幕高频词: [观众在讨论什么]
可复制点: [我们直接能学的]
差异化空间: [我们能做得更好的]
```

### 4.1 竞对信号速查

| 信号 | 含义 |
|------|------|
| 弹幕说「AI做的？」 | 质量差，被识破 |
| 弹幕说「这是AI？」 | 质量好，被惊艳 |
| 评论区催更 | 选题命中需求 |
| 收藏>点赞 | 实用价值高 |
| 投币>点赞 | 极高认可 |

## 五、内容日历策略

### 5.1 发布节奏

| 阶段 | 频率 | 目的 |
|------|------|------|
| 冷启动（前 5 集） | 隔天 1 集 | 测试选题 + 积累初始粉丝 |
| 验证期（6-15 集） | 每周 3 集 | 稳定产出 + 培养追更习惯 |
| 成熟期（16 集+） | 每周 2 集 | 质量优先 + 爱发电变现 |

### 5.2 系列化策略

```
主系列（周更 2 集）→ 核心 IP
  ├─ 番外（月更 1 集，爱发电专属）
  ├─ 角色日常（抖音短视频，15 秒）
  └─ 联动（与其他 AI UP 主合作）
```
